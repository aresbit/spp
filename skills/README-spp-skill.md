# spp-skill

`skills/spp-skill` now targets **external application developers** who want to build projects on top of SPP.

## Scope

- New app scaffolding with modern-c-makefile style
- Practical use of SPP ownership/containers/result/concurrency/I-O APIs
- std-to-SPP migration guidance for existing codebases
- Build, test, and troubleshooting flow for app teams

## Key files

- Skill entry: `skills/spp-skill/SKILL.md`
- Integration reference: `skills/spp-skill/references/project-integration.md`
- Skill metadata: `skills/spp-skill/agents/openai.yaml`

## Invocation

Use `$spp-skill` when the task is: "how do I build my app with SPP" rather than "how to develop SPP internals".
